
## az dataprotection backup-vault create ##

Deletes a BackupVault resource from the resource group.
### Parameters ###
  - --resource-group -g
  - --vault-name
  - --yes -y

### Examples ###

#### Create a backup Vault ####
Command
```
az dataprotection backup-vault delete -g sarath-rg --vault-name sarath-vault --yes
```
Output
